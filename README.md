Steven crawler
========
It crawls the URLs of Steven Universe from [bravebold.tistory.com]. Requires
Node.js v7.0+

![Pearl](https://i.imgur.com/3BgE5BA.png)

```bash
./run | tee output

# If you're using node < v7.6
node --harmony-async-await run | tee output
```
